# Ganlin Chen - Personal Website

Academic portfolio for Ganlin Chen, a computational materials scientist at the
University of Michigan.

## Publishing with GitHub Pages

The site uses plain HTML and CSS and requires no build step. In the repository:

1. Open **Settings → Pages**.
2. Under **Build and deployment**, select **Deploy from a branch**.
3. Choose the `main` branch and the `/ (root)` folder.
4. Save.

GitHub Pages may take several minutes to publish the first deployment.
